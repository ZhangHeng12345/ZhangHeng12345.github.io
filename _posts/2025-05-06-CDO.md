# cdo使用记录
## 1.插值：
```
cdo remapbil,r360x180 in.nc out.nc
```
## 2. 选取变量名，计算集合平均，计算月气候态并插值
```
cdo -remapbil,r192x94 -ymonmean -ensmean -selname,huss huss_Amon_AWI-ESM-1-1-LR_lgm_r1i1p1f1_gn_39* huss_AWI.nc
```
